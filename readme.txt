Прежде всего установить yarn
В консоль: npm i -g yarn
Выключите все консоли и редакторы кода после установки

1. Распакуйте архив в одну папку

2. В папке с распакованным архивом пропишите в консоль: yarn 

3. Чтобы зыпустить код пропишите в консоль: yarn start


Редактировать: src/index.ts

Писать строго на typescript
